# Computational Physics Projects

Welcome to my repository showcasing computational physics projects. These are created out of curiosity and to gain valuable experience for potential research placements.

## About

I'm a Year 11 student learning computational physics by implementing classical mechanics simulations (currently self-studying from R. Shankar's *Fundamentals of Physics 1*). Other areas of physics may be added in the future.

---

## Projects

### 1. Projectile Motion
Simulates projectile trajectories under gravity using kinematic equations.

```bash
cd projectile-motion
python projectile.py
```

### 2. N-Body Gravity
Simulates the motion of multiple bodies interacting through Newtonian gravity in 2D. Users can define:

- Number of particles
- Masses
- Initial positions and velocities

Trajectories are visualized in real-time using matplotlib.

```bash
cd n-body-gravity
python n_body.py
```

### Simple Harmonic Oscillator
Simulates the motion of a particle oscillating in accordance to F=-kx and ignoring friction/air resistance. Users define:

- Spring constant
- Mass of particle
- Initial extention

Oscillation shown in real time using matplotlib.

```bash
cd simple-harmonic-oscillator
python simpleoscillator.py
```

### Diffusion simulation
Simulates diffusion with 10000 "walkers" moving a step in a random direction with each time step. Users shown a heatmap of particle positions and graph of diffusion constant against time with ideal diffusion constant (0.25) shown in red.

This is all shown in real time using matplotlib

```bash
cd random-walkers
python random_walkers.py
```
